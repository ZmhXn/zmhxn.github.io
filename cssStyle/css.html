# (1) Animate.css (opens new window)

基础用法

    引入：

           <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/3.7.0/animate.min.css">

        <div id="box" class="animated bounce"></div>

# (2) Aos-超赞页面滚动元素动画 (opens new window)

vue中引用

    安装：
        npm install aos --save
    然后在vue文件的main.js中引入：
        import AOS from 'aos'
        import 'aos/dist/aos.css'
        AOS.init({
             offset: 200, //是立刻触发动画还是在指定时间之后触发动画
             duration: 600, //动画持续时间
             easing: 'ease-in-sine', //动画的easing动画效果
             delay: 100, //动画的延迟时间
        }); //初始化 （可以手动配置一些参数）
    例：
        <div data-aos="flip-left" data-aos-delay="100"></div>

# 利用 CSS 穿透覆盖默认样式

常见发生场景：假如我们需要通过 input，type="file"来上传文件，而这个 input 的默认样式，可以说是非常地丑。所以我们希望通过一张图片，与这个 input 大小一样，位置一致地盖在上面。这个时候，显然，这个时候点击图片，input 是不会起作用的。就是因为 img 隔绝了 click 的穿透，而我们希望的是，这个 img 只是视觉上遮挡了 input 的样式，但是点击的时候还是点击到 input。所以，只要让 img 可穿透即可。

    img {
        pointer-events: none;
    }

# 文本溢出处理

移动设备相对来说页面较小，很多时候显示的一些信息都需要省略部分。最常见的是单行标题溢出省略，多行详情介绍溢出省略。现在都用框架开发了，这种建议需求建议形成一个基础组件，方便快捷。

    /* 单行 */
    .single {
        overflow: hidden;
        white-space: nowrap;
        text-overflow: ellipsis;
    }
    /* 多行 */
    .more {
        display: -webkit-box !important;
        overflow: hidden;
        text-overflow: ellipsis;
        work-break: break-all;
        -webkit-box-orient: vertical;
        -webkit-line-clamp: 2; //指定行数
    }
